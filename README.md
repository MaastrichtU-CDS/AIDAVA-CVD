# AIDAVA-CVD
Supplementary materials for AIDAVA CVD case

1. DTS files with AIDAVA sample data for mapping into source and personal health knowledge graphs
2. SPARQL query examples from the personal health knowledge graphs to extract necessary metrics for SMART risk score calculation
3. Mapping examples with RDF triples
4. Data elements and computational algorithm (R script) for SMART risk score calculation
5. Questionnaire example to evaluate user medical literacy and digital skill levels
